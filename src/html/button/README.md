# Button

## Useage
Import the component
```
import { Button } from '@leetajz/vue-components'
```

In your template
```
<Button
    :background=""
    :size=""
    :icon=""
    :text=""
/>
```

## Props
| Props        | Type     | Required  | Default       |
|:-------------|:---------|:----------|:--------------|
| background   | String   | false     | transparent   |
| size         | String   | false     | md            |
| icon         | String   | false     |               |
| text         | String   | false     | Click me      |
