# ButtonNuxt

## Useage
Import the component
```
import { ButtonNuxt } from '@swegaming-ab/vue-components'
```

In your template
```
<ButtonNuxt
    :background=""
    :size=""
    :icon=""
    :link=""
    :text=""
/>
```

## Props
| Props        | Type     | Required  | Default       |
|:-------------|:---------|:----------|:--------------|
| background   | String   | false     | transparent   |
| size         | String   | false     | md            |
| icon         | String   | false     |               |
| link         | Object   | true      |               |
| text         | String   | false     | Click me      |
